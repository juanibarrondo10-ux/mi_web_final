Descripción del Proyecto:

Este proyecto es una simulación de un sitio web de comercio electrónico para una librería

Principales puntos:

**Renderizado Dinámico de Productos:** Los productos se cargan a partir de datos estáticos en JavaScript `productosLibreria` y se insertan de manera dinámica en el HTML.
* **Carrito de Compras Persistente:** Utiliza `localStorage` para guardar los productos seleccionados por el usuario, manteniendo el estado del carrito después de recargar la página.
* **Funcionalidad Completa del Carrito:**
    * Agregar productos al carrito.
    * Eliminar productos individualmente.
    * Vaciar el carrito completamente.
    * Contador dinámico de productos.
* **Validación de Formulario:** Implementación de JavaScript para la validación del formulario de contacto (campos requeridos y formato de correo electrónico).
* **Diseño Responsivo:** Se utiliza CSS Flexbox para la maquetación de las tarjetas, asegurando que se adaptean correctamente a diferentes tamaños de pantalla y que el contenido quede alineado.

## Tecnologías Utilizadas

* **HTML5** 
* **CSS3** 
* **JavaScript**
